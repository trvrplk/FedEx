FedEx -- Easy ActionMailer

FedEx is a small DSL-like gem that makes using the awesome ActionMailer.

For the sake of re-usability, you must still define a method to create new mailer. However, this code with ActionMailer:

  class HelloWorld < ActionMailer::Base
  delivers_from 'foo@example.com'

  def hello(recipient)
    @recipient = recipient
    mail(:to => recipient,
         :subject => "O HAI!")
  end
end

Becomes this with FedEx:

  class HelloWorld < ActionMailer::Base
    delivers_from 'foo@example.com'

    def hello
      recipients 'bar@example.com'
      subject "O HAI!"
